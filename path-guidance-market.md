<p><ul><li>Magnetic Bar</li><li>Optical Guidance (OG)</li><li>Magnetic Tape</li><li>Laster Guidance</li><li>Indoor Global Positioning System</li><li>Attitude Heading Reference System</li></ul></p>
<p><p>Path guidance systems are technologies used to guide automated vehicles along pre-determined routes. Different types of path guidance markets include Magnetic Bar, Optical Guidance (OG), Magnetic Tape, Laster Guidance, Indoor Global Positioning System, and Attitude Heading Reference System. Magnetic Bar systems use magnets embedded in the floor to guide vehicles, while OG systems use cameras to follow lines or markers. Magnetic Tape systems use magnetic strips on the floor, Laster Guidance uses lasers for navigation, Indoor Global Positioning System uses satellites, and Attitude Heading Reference System uses sensors for precise positioning.</p></p>
